# vim安装插件依赖

## 安装vim-Plug

```shell
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## coc.nvim依赖

```shell
brew install node
sudo npm i -g neovim yarn
```

## vim-instant-markdown依赖

```shell
sudo npm install instant-markdown-d
```
